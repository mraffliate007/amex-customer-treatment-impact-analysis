# Dataset Info

**Name:** Marketing Campaign
**Source:** [Kaggle - rodsaldanha/arketing-campaign](https://www.kaggle.com/datasets/rodsaldanha/arketing-campaign)
**File:** `marketing_campaign.csv`

## How to use
1. Download the CSV from the Kaggle link above (free Kaggle account required)
2. Place it in this `data/` folder, or load directly into your Python environment

## Why it's not committed directly
This raw dataset is sourced from a public Kaggle dataset and is linked here rather than duplicated in this repo, in line with standard data project practice.

## Columns (summary)
| Column | Description |
|---|---|
| ID | Customer identifier |
| Year_Birth | Customer birth year |
| Education | Education level |
| Marital_Status | Marital status |
| Income | Yearly household income |
| Kidhome / Teenhome | Number of children/teens at home |
| Dt_Customer | Date customer was enrolled |
| Recency | Days since last purchase |
| MntWines, MntFruits, MntMeatProducts, etc. | Amount spent on each product category in last 2 years |
| NumDealsPurchases | Number of purchases made with a discount |
| NumWebPurchases, NumStorePurchases, NumCatalogPurchases | Purchases by channel |
| AcceptedCmp1–5 | Whether customer accepted offer in campaign 1–5 (1 = yes, 0 = no) |
| Response | Whether customer accepted the offer in the last campaign |
